## 🐯 NeverSleepTiger (PC 절전 모드 방지 도구)
NeverSleepTiger는 윈도우 커널 신호를 직접 제어하여 시스템이 절전 모드로 진입하는 것을 방지하는 초경량 유틸리티입니다.

## ⚡ 사용법
파일 실행 시, 트레이 아이콘 리스트에 등장함.
1. 클릭 1번으로 상태 전환(활성화, 비활성화)
2. 타이머 설정 가능

<br>
## 🚀 주요 특징 (Key Features)
- ⚡ 가상 키 입력 방식이 아닌 윈도우 공식 전원 API(SetThreadExecutionState)를 사용하여 보안 프로그램 간섭 없이 확실하게 작동합니다.  
- 🔋 리소스 점유율 0% 지향: 30초마다 1회 신호만 갱신하며 대기합니다. CPU 점유율은 사실상 0%이며, 메모리 사용량은 메모장 수준으로 앱플레이어 대비 0.01% 미만입니다.  
- 📂 단일 파일(Portable): 별도의 설치나 외부 라이브러리 없이, EXE 파일 하나로 어디서든 즉시 실행 가능합니다.  
- 🖱️ 트레이 상주: 시계 옆 트레이 아이콘에 숨어서 작동하며, 우클릭 메뉴로 간편하게 제어합니다.  
- ⚠️ 주의 사항 (Cautions)  
- 수동 종료 필요: 별도의 예약 종료 기능이 없습니다. 프로그램을 종료하지 않고 퇴근할 경우 밤새 PC가 켜져 있을 수 있으니 주의해 주세요.
